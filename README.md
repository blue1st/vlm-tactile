# VLM-Tactile

VLM-Tactile は、Vision Language Model (VLM) を利用してデスクトップ画面を解析し、マウス操作やキーボード入力を自動化するためのデスクトップエージェントツールです。

## 概要

このプロジェクトは、AI（特にマルチモーダルモデル）が視覚情報を基にコンピューターを操作する可能性を探るための実験的なプラットフォームです。
現在は主に **Ollama** を通じたローカルモデル（Gemma 4 など）の使用を想定していますが、将来的に様々なモデルへの対応を考慮した設計になっています。

## 主な機能

- **画面キャプチャ & 解析**: 指定した画面範囲をキャプチャし、VLMに送信して次のアクションを決定させます。
- **多様なアクション**: 
  - クリック / 右クリック / ダブルクリック
  - ドラッグ＆ドロップ
  - スクロール
  - キーボード入力（WASD, Enter, Spaceなど）
- **リアルタイム・フィードバック**: AIの思考プロセス（Analyzing...）や実行したアクションをUI上で視認できます。
- **エリア選択機能**: 画面全体ではなく、特定のウィンドウやゲーム画面の一部だけを対象にすることが可能です。
- **VLMベンチマーク**: 搭載されたベンチマーク機能により、モデルの推論能力をテストできます。

## 必要条件

- **OS**: macOS / Windows / Linux (Electronが動作する環境)
- **Node.js**: v18以上推奨
- **Ollama**: ローカルでマルチモーダルモデル（`gemma4` 等）が動作していること

## セットアップ

1. **リポジトリのクローン:**
   ```bash
   git clone <repository-url>
   cd vlm-tactile
   ```

2. **依存関係のインストール:**
   ```bash
   npm install
   ```

3. **起動:**
   ```bash
   npm start
   ```

## 使い方

1. **Ollamaの起動**: PCでOllamaが動作しており、モデル（例: `gemma4:it`）がプルされていることを確認してください。
2. **モデル選択**: UI上の「Model」セレクトボックスから使用するモデルを選択します。
3. **範囲指定**: プレビューエリアでドラッグして、AIに認識させたい範囲を選択します。
4. **プロンプト設定**: 「Additional User Instructions」に、「赤いボタンをクリックして」「キャラクターをWASDで移動させて」などの具体的な指示を入力できます。
5. **開始**: 「Start Automation」ボタンを押すと、定期的に画面解析と操作が実行されます。

## 注意事項

- 本ツールはマウスやキーボードを直接操作します。不測の事態に備え、常に「Stop」ボタンや緊急停止（Ctrl+C等）ができる状態で使用してください。
- 解析精度や動作の正確性は使用するVLMの性能に依存します。
